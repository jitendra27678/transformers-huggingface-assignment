# Understanding Transformers and Hugging Face

This project demonstrates the Transformer architecture and practical NLP applications using the Hugging Face Transformers library.

## Topics Covered
- NLP Fundamentals
- Transformer Architecture
- BERT Model Card Analysis
- Text Summarization using BART
- Language Translation using MarianMT

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Google Colab
